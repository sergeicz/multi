# Kinetic Terminal // ОПЕРАТОР_УЗЛА

Этот проект представляет собой стилизованный терминал пожертвований и профиль пользователя в стиле Kinetic Monolith.

## Особенности
- **Дизайн**: Бруталистический индустриальный стиль с элементами "стеклянного" морфизма.
- **Анимации**: Эффект набора текста, помехи (glitch), пульсация и сканирующие линии.
- **Функционал**: Выбор криптовалютных адресов с динамическим выводом в терминал и генерацией QR-кодов.
- **Адаптивность**: Полная поддержка мобильных устройств.

## Деплой на GitHub Pages
1. Загрузите файлы в репозиторий GitHub.
2. Перейдите в **Settings** > **Pages**.
3. Выберите ветку `main` и папку `/ (root)`.
4. Нажмите **Save**.
